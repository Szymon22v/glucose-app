Opis działania testera

Tester analizuje dane JSON zawierające wynik glukozy. Program wyszukuje parametr GLU, odczytuje wartość, jednostkę i zakres referencyjny, a następnie przypisuje flagę:

  • L — wynik poniżej normy,
  • N — wynik w normie,
  • H — wynik powyżej normy.

Dla wartości zapisanych w mmol/L tester przelicza wynik na mg/dL. Moduł działa w przeglądarce i służy wyłącznie do testowania plików JSON.
